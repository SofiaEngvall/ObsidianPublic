
Go to `/usr/share/applications`

sudo nano program-name.desktop

add this
```
[Desktop Entry]
Version=1.0
Type=Application
Name=<name-of-the-app>
Exec=<path-to-executable-file>
Icon=<path-to-icon-file>
Categories=<list-of-;-separated-categories>
```
