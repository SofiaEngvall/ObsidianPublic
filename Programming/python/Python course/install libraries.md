### virtual environments

install it
`pip install virtualenv`

make a dir for your env
`mkdir virtual-env`
`cd virtual-env`

start and activate
`python3 -m venv env`
`source env/bin/activate`
starts fresh, no modules installed

check with
`which python3`
`which pip`
`pip freeze` lists installed modules
